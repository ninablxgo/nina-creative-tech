Realtime participatory interface & networked visual feedback

For this project, I collaborated with the queer music collective Club Reverie to develop an immersive projection environment for the clubbing space of La Communale, Saint-Ouen. Using MadMapper, Resolume and TouchDesigner, we designed a live visual system combining projection mapping and real-time audience interaction. One of the main challenges was balancing two parallel needs: creating a stable, responsive chat interface mapped directly into the performance space - with live reset capabilities - while simultaneously producing continuous generative visuals throughout the event.

This project explores lightweight real-time interaction between a mobile audience interface and a generative visual system. A custom web client connects directly to a public WebSocket chat server, enabling participants to send live inputs without relying on a dedicated backend infrastructure. Audience messages are transmitted as structured JSON data and parsed in TouchDesigner to drive visual behaviours during live performance.

Technical Structure

The system was designed as a lightweight real-time architecture enabling audience participation without requiring a dedicated backend. A custom web interface acts as a WebSocket client, joining a shared chat channel where messages are transmitted as structured JSON data. Inside TouchDesigner, incoming data is parsed and mapped to visual parameters, allowing live modulation of the projection environment. Special attention was given to connection stability, nickname sanitization and UX simplicity to ensure reliable public interaction during performance conditions.
