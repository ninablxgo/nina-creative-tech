Animal-Machine — Embodied Hybrid Agent

Embodied voiceinterface & hybrid machine presence

Animal-Machine explores the construction of an embodied digital agent distributed between physical robotics and virtual simulation. The project consists of a 3D-printed autonomous creature mounted on wheels, paired with a Unity-based virtual counterpart sharing the same behavioural identity. Rather than functioning as a conventional robot, the system operates as a hybrid interface oscillating between material presence and software representation.

Animal-Machine, 3D print, Arduino, speech recognition system, 2023

System Logic

The physical body is controlled via Arduino and programmed to navigate space autonomously using an obstacle detection sensor. Its movement logic is intentionally non-deterministic, generating unpredictable behavioural patterns in space. Inside the body, a screen displays a Unity environment hosting a virtual version of the same entity.

A predefined voice command system allows verbal interaction. When specific phrases are detected, scripted responses are triggered in Unity, producing the illusion of dialogue between human and machine.

Interaction Demonstration

Animal-Machine investigates the boundaries between artificial agency and performative illusion. By splitting identity across hardware mobility and scripted dialogue, the project examines how minimal rule-based systems can generate perceived intentionality. The machine does not understand language; it performs understanding. This ambiguity between response and cognition forms the conceptual core of the work.
