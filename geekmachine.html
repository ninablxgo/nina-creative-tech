GeekMachine — Biometric Game Interface

Game data to ecological feedback loop

GeekMachine is a hybrid sculpture exploring the relationship between competitive online behaviour and ecological systems. The installation retrieves historical match data from the Riot Games API, looping through a player’s recent games to generate asynchronous interactions between past gameplay events and a living system.

GeekMachine, 3D printed sculpture with Raspberry Pi–driven API interaction and environmental feedback system, 2024

System Logic

Riot API (match history) → Python Parser → Raspberry Pi GPIO → Pump / LED / Audio
Moisture Sensor → Feedback Loop → Watering Override

The system retrieves the latest League of Legends matches through the Riot Games API, looping through the twenty most recent games. Each gameplay event triggers specific physical responses:

  • If a match is won, a water pump is activated to irrigate the plant.
  • A soil moisture sensor prevents activation when the environment is already hydrated, introducing a biological constraint to the reward system.
  • Enemy kills trigger a LED flash and a low growl sound played through speakers.

This layered logic transforms competitive metrics into a fragile ecosystem where digital success does not always translate into physical action.

Technical Implementation

A Raspberry Pi runs a Python environment connecting to the Riot API while simultaneously controlling hardware outputs. Game events are parsed in real-time and mapped to GPIO triggers driving the pump relay, LED feedback and audio playback.

The moisture sensor introduces a feedback loop preventing overwatering, allowing the living system to resist purely algorithmic control. This constraint became a central conceptual element, shifting the sculpture away from direct automation toward negotiated interaction.

Research Direction

GeekMachine investigates how data-driven systems can produce intimate relationships between players and non-human entities. By translating invisible gameplay metrics into tangible environmental consequences, the project questions the logic of optimisation embedded in digital culture. The sculpture becomes a mediator between performance, care and ecological balance.
